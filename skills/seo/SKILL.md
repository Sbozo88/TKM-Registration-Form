# SEO Skill

This skill provides a checklist for ensuring good SEO.

## Instructions
- [ ] **Title Tag**: Each page must have a unique, descriptive `<title>`.
- [ ] **Meta Description**: Add `<meta name="description" content="...">`.
- [ ] **H1 Tag**: Ensure exactly one `<h1>` per page.
- [ ] **Alt Text**: Add `alt` attributes to all `<img>` tags.
- [ ] **Semantic HTML**: Use `<header>`, `<main>`, `<footer>`, `<nav>`, `<article>`, etc.
- [ ] **Open Graph**: Add `og:title`, `og:description`, `og:image` meta tags for social sharing.
